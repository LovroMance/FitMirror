# FitMirror

AI 动作教练 · 训练日历 · PWA 健身应用

## 目录结构

详见 docs/项目代码目录结构.md

## 快速开始

1. 运行 `scripts/setup.sh` 安装依赖
2. 前端开发：`cd frontend && npm run dev`
3. 后端开发：`cd backend && npm run dev`

## 主要技术栈

- 前端：Vue3 + Vite + TypeScript + Pinia + Element Plus
- 后端：Node.js + TypeScript + Express + MySQL

## 代码规范

- ESLint、Prettier、Stylelint、Husky、lint-staged
- 详见 docs/开发指南.md
