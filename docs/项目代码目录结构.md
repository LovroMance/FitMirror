# FitMirror 项目代码目录结构

文档版本：v2.0  
最后更新：2026-03-31

本文档只描述当前仓库真实存在且仍在使用的目录与模块。

## 一、仓库结构

```text
FitMirror/
├── backend/                     # Express + Prisma 后端
├── docs/                        # 项目文档
├── frontend/                    # Vue3 + Vite 前端
├── scripts/                     # 根目录辅助脚本
├── UI-design/                   # 设计源文件
├── .env.example
├── README.md
└── package.json
```

## 二、frontend 当前结构

```text
frontend/
├── public/
│   ├── data/
│   │   └── exercises.json       # 动作库静态数据
│   ├── favicon.ico
│   └── manifest.json            # PWA manifest
├── src/
│   ├── api/                     # 接口封装
│   ├── assets/styles/           # 全局样式
│   ├── components/
│   │   ├── common/              # 通用组件
│   │   └── layout/              # 布局组件
│   ├── config/                  # 页面配置常量
│   ├── db/                      # Dexie 数据库
│   ├── repositories/            # 本地数据访问封装
│   ├── router/                  # 路由配置与守卫
│   ├── store/                   # Pinia 状态管理
│   ├── types/                   # 类型定义
│   ├── utils/                   # 工具函数
│   ├── views/                   # 页面视图
│   ├── App.vue
│   └── main.ts
├── vite.config.ts
├── vitest.config.mjs
├── tsconfig.json
└── package.json
```

### 前端当前核心页面

- `Home.vue`：首页与主导航
- `Login.vue`、`Registration.vue`：认证页
- `PlanGenerator.vue`：计划生成与最近计划恢复
- `WorkoutSession.vue`：真实训练会话
- `WorkoutLog.vue`：热图、详情、手动补录
- `Exercises.vue`：动作库筛选与详情
