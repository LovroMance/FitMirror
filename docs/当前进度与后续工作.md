## 2. 当前系统状态（可用能力）

### 账号与鉴权
- 已具备：注册、登录、`/me`、JWT 持久化、路由守卫。

### 计划生成
- 接口：
  - `POST /api/plans/generate`
  - `GET /api/plans/generate/stream`
- 策略：DeepSeek 优先，失败自动模板回退。
- 前端：可展示来源标签（`deepseek/template`），支持流式进度反馈。

### 本地数据与可视化
- Dexie 三表可用：`plans`、`workout_records`、`settings`。
- 训练记录热图可按日聚合显示，支持点击日期看详情。

### 动作库
- 静态数据源可用（`public/data/exercises.json`）。
- 支持关键词 + 部位 + 难度 + 器械筛选与详情弹层。
