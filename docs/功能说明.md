# FitMirror 功能说明

文档版本：v4.0  
最后更新：2026-03-31

本文档只描述产品功能清单与功能边界，不记录开发进度。

## 一、账号与鉴权

- 用户注册
- 用户登录
- 获取当前用户信息 `GET /api/auth/me`
- JWT 登录态持久化
- 路由守卫与登录态恢复

## 二、训练计划生成

- 稳定通道：`POST /api/plans/generate`
- 流式通道：`GET /api/plans/generate/stream`
- DeepSeek 优先，失败时回退模板计划
- 前端展示流式状态反馈
- 展示计划生成来源
- 最近计划保存、恢复、删除

## 三、训练会话

- 从计划页进入训练会话页
- 按动作顺序推进训练
- 支持暂停、继续、结束
- 完成训练后自动写入训练记录

## 四、训练记录

- 最近六周热图展示
- 训练天数、总时长、连续打卡统计
- 查看某日训练详情
- 手动补录入口

## 五、动作库

- 读取静态动作数据 `frontend/public/data/exercises.json`
- 关键词筛选
- 按部位、难度、器械筛选
- 查看动作详情
- 从计划页跳转动作库并带入预筛选
- 动作名归一化匹配

## 六、本地数据层

- Dexie 管理 `plans`
- Dexie 管理 `workout_records`
- Dexie 管理 `settings`
- 最近计划从 `localStorage` 向 Dexie 的一次性迁移

## 七、功能边界

- 后端不存储训练记录
- 训练会话不包含摄像头识别
- 产品只支持移动端 PWA 方向
- PWA 只包含 `manifest.json` 基础配置，不包含完整离线能力
